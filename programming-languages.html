Language categories

Some languages are designed for web development, others for systems programming, and some for rapid prototyping. Knowing the category helps narrow your focus.

Web languages

HTML, CSS, and JavaScript power the front-end experience. Together they build interactive interfaces that run in the browser.

General purpose

Languages like Python, Java, and Go can be used for many different tasks, from automation to server-side applications.

Systems languages

Rust and C++ give you low-level control and performance for building efficient software and infrastructure.
